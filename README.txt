1) Извлечь архив (source.zip) "В текущую папку" (то есть в папку со скриптом).

2) В винде, в файле (link2pic.bat), потребуется лишь заменить ("....path....") на правильный путь к папке со скриптом (link2pic.py)

3) Чтобы запустить скрипт, достаточно открыть bat-файл.

**3) Не стОит забывать, что для правильной работы скрипта потребуется vpn connection (по-другому никак :-{ ), иначе будет ошибка.
- для винды проще всего юзать softether с пакетом VPN Gate
- для линя лучше юзать либо самопал (мб, что-нибудь платное), либо этот гайд (*тоже просто подключение к впн гейт, но по опен впн без заморочек):
https://gist.github.com/Lazza/bbc15561b65c16db8ca8

Очевидно, что для запуска скрипта под линем bat-файл не требуется.
